Vue implements a content distribution API using the element to serve as distributer for content.

This allows you to compose components like this: When the component renders, will be replaced by “Your Profile”. Slots can contain any template code, including HTML

types of slots

named slots:they are having a special attribute called name which is used to allocate multiple slots when necessary

unnamed slots:we cannot use multiple slots in unnamed slots and any variable within these will be used within the scope
